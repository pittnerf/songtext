Put one PDF per song in this folder (`songs_pdfs/`).

You can also use the alternate folder name `songs_pdf/`.

Naming (recommended):
  01_amazing_grace.pdf
  02_how_great_thou_art.pdf
  03_silent_night.pdf

The leading number becomes the song number shown in the controller.
The rest of the filename becomes the song title.

Then run from the repo root:
  python3 9_songtext/scripts/build_songs.py
