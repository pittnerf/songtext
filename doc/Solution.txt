Set up Firebase (free):

Create a project at console.firebase.google.com
Enable Realtime Database
Copy the web app config into 9_songtext/public/js/config.js
Test locally (server is running on port 8765):

http://localhost:8765/controller.html
http://localhost:8765/viewer.html
Deploy the contents of 9_songtext/public/ to GitHub Pages, then open qr.html on the live URL.

Full setup details, Firebase rules, and a concert-day checklist are in 9_songtext/README.md.

Note: The songs_pdf/ folder is empty right now, so the app uses 3 sample songs. Once you drop in your PDFs and run the build script, those will replace the samples.

Deploy size too large: --jpeg-quality 80
Example:

python3 9_songtext/scripts/build_songs.py --scale 2.5 --jpeg-quality 85
